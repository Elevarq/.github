# Elevarq

PostgreSQL tools for engineering teams.

Website: https://elevarq.com

## Repositories

- [Arq-Signals](https://github.com/Elevarq/Arq-Signals) — read-only PostgreSQL
  telemetry collector. Open source.
- [pgAgroal](https://github.com/Elevarq/pgAgroal) — production-ready container
  distribution of pgagroal, a high-performance PostgreSQL connection pooler.
  Open source, BSD-3.

## Commercial

[Arq](https://elevarq.com/products/arq) is our commercial PostgreSQL
intelligence platform. Diagnostic reasoning over monitoring — answering *why*
something is happening and what to do about it.
